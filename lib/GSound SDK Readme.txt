#### GSound SDK Readme - Version Beta 5
#############################################

*** SDK Contents ***
./Mac OS X/GSound.a : compiled static library of GSound, universal binary.
./Mac OS X/include : folder containing all GSound header files
./Windows/GSound.a : compiled static library of GSound.
./Windows/include : folder containing all GSound header files
./Example 1 - Simple : folder containing a simple example application and source code.
./Example 2 - Complex : folder containing a complex example application and source code.


*** System Requirements ***
Mac OS 10.5 or later, PPC or Intel, 32 or 64 bit.
Windows Vista or later, 32 bit.


*** Documentation ***
Documentation is online at the GSound main website:
http://gamma.cs.unc.edu/GSOUND/
